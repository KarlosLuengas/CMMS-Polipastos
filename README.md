# CMMS-Polipastos
## Sistema para la gestión de operaciones del mantenimiento de polipastos 

<div>
  Este sistema es de carácter privado, por lo que no compartiré detalles específicos ni recursos del proyecto. La siguiente información tiene como objetivo describir la estructura general, los objetivos y las tecnologías empleadas en el desarrollo de este sistema CMMS.
</div>

## Objetivo 

<div>
    Este proyecto fue desarrollado principalmente para mejorar la eficiencia en el mantenimiento de polipastos, grúas y troles, optimizando el rendimiento del proceso y ofreciendo una mejor impresión a los clientes. De esta manera, se busca reducir gastos y tiempos de ejecución, permitiendo realizar los servicios con mayor rapidez. Además, el sistema garantiza que el cliente esté siempre informado sobre el estado de su equipo, fortaleciendo la confianza y la transparencia en cada servicio.
</div>

## Estructura

<div>
  El sistema es completamente privado y cuenta con múltiples restricciones según el tipo de usuario. Incluye mecanismos de autenticación y diversas medidas de seguridad para garantizar el acceso controlado y la protección de la información.
  
  ### Autenticación
    - Login / Logout
    - Usuarios y permisos (Admin, Técnico, Cliente)
    - Restricciones según rol
  ### Accesos
    - Vista de tabla de tiempos de los equipos
    - Usuarios
    - Directorio de marcas, modelos, parámetros y componentes
    - Galería
    - Reportes
  ### Reportes
    - Reportes de la recepción hasta la salida del equipo
    - Generación automática de reportes de Inspección y de salida
    - Descarga en PDF los reportes
    - Alimentados por los datos del directorio
  ### Órdenes de trabajo
    - Asignación a técnicos
    - Estatus (pendiente, en proceso, terminado)
    - Evidencia fotográfica
    - Firma del cliente
    - Alertas y notificaciones
  ### Clientes
    - Perfil del cliente
    - Equipos asignados
    - Panel para ver el estado de sus equipos

</div>
